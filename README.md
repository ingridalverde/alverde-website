# Alverde Coaching — Website

Four-page HTML/CSS website for alverdecoaching.com

## Files

- `index.html` — Home page
- `about.html` — About Ingrid
- `services.html` — Services & pricing (Jumpstart, Spark, Roots, Flourish)
- `contact.html` — Contact form
- `styles.css` — Shared stylesheet (all brand colors, fonts, layout)
- `headshot.jpg` — **Add your headshot photo here** (rename your photo to headshot.jpg)

## How to add your headshot

1. Save your headshot photo as `headshot.jpg`
2. Place it in this folder alongside the HTML files
3. It will automatically appear on the Home and About pages

## How to publish on GitHub Pages

1. Create a free account at github.com if you don't have one
2. Click **New repository** — name it `alverdecoaching.com` or `alverde-website`
3. Upload all files in this folder to the repository
4. Go to **Settings → Pages**
5. Under Source, select **Deploy from a branch → main → / (root)**
6. Click Save — your site will be live at `yourusername.github.io/alverde-website`

## How to connect your alverdecoaching.com domain

1. In GitHub Pages settings, enter `alverdecoaching.com` under Custom Domain
2. Log into wherever your domain is registered
3. Add these DNS records:
   - A record: 185.199.108.153
   - A record: 185.199.109.153
   - A record: 185.199.110.153
   - A record: 185.199.111.153
4. DNS changes take up to 24 hours to propagate

The site uses Google Fonts (Cormorant Garamond + DM Sans) loaded via CDN.
No installation needed — requires internet connection to display correctly.
